ING Support

最后更新：2026-06-15

1. 关于 ING

ING 用于帮助用户在出发前参考地点现场状态。

2. 当前版本使用说明

  • 当前版本不需要账号。
  • 当前版本不需要测试账号。
  • 当前版本不进行真实用户数据写入。
  • 当前版本暂不进行真实图片上传。
  • 当前版本不要求真实定位权限。

3. 联系支持

如需反馈问题、咨询隐私事项或提出建议，请发送邮件至：ingsupport@163.com。

反馈时建议提供以下信息，方便定位问题：

  • 设备型号
  • iOS 版本
  • App 版本
  • 问题页面
  • 截图

4. 常见问题

  1. 是否需要登录？
    当前不需要。
  2. 是否会上传我的发布内容？
    当前版本不进行真实用户数据写入。
  3. 图片功能是否可用？
    当前版本暂不进行真实图片上传。
  4. 是否需要定位权限？
    当前版本不要求真实定位权限。
  5. 如何反馈问题？
    通过支持邮箱 ingsupport@163.com 联系。

English Summary

ING helps users reference place status before heading out. The current version does not require an account or a test account, does not write user data, does not perform real image uploads, and does not require real location permission.

Support contact: ingsupport@163.com
