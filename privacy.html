ING Privacy Policy

最后更新：2026-06-15

1. 适用范围

本隐私政策适用于 ING。ING 用于帮助用户在出发前参考地点现场状态。

2. 当前版本的数据边界

  • 当前版本不需要登录。
  • 当前版本不上传用户发布内容。
  • 当前版本不进行用户数据写入。
  • 当前默认数据源为本地数据 / 本地示例数据。
  • 当前不接 账号登录服务。
  • 当前不运行 远程数据库操作。
  • 当前不建远程数据库。
  • 当前不接用户发布内容上传或保存。
  • 当前不接广告。
  • 当前不接第三方分析 SDK。
  • 当前不接推送。
  • 当前不进行真实图片上传。

3. 账号与登录

当前版本不要求用户创建账号，也不要求用户登录。若未来启用账号体系，ING 将在启用前更新本隐私政策，并说明账号数据的使用方式。

4. 用户内容

当前版本不会把用户发布内容上传到云端，也不会进行真实用户数据写入。App 内的发布、反馈、图片入口等能力仅按当前版本边界展示，不代表已经启用真实上传。

5. 定位、图片、通知与分析

当前版本不要求真实定位权限，不进行真实图片上传，不接推送，不接广告，也不接第三方分析 SDK。

6. 未来功能变化

如未来启用远程公开地点数据读取、账号能力、用户内容上传、图片上传、通知或其他数据能力，ING 将在相关功能上线前更新隐私政策，并在必要时提供用户选择、删除、咨询或反馈路径。

7. 联系我们

如需咨询、删除请求或反馈隐私问题，请通过支持邮箱联系：ingsupport@163.com。

English Summary

ING currently does not require sign-in, does not upload user-generated content, does not write user data, does not use ads, does not use third-party analytics SDKs, does not enable push notifications, and does not perform real image uploads. If future versions enable remote public place data, account features, or user content uploads, this privacy policy will be updated before release.

Contact: ingsupport@163.com
